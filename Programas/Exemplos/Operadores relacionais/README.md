# Operadores relacionais


* True ou False 

  * 5 > 4  : True 
  * 5 < 4  : False
  * 5 <= 4 : False
  * 5 >= 4 : True
  * 3 != 2 : False
  * 3 == 3 : True
  * 2 == '2' : False
